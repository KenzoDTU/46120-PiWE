## In-class quiz

In class in Week 13 (either in person or online), there will be a small quiz to very a few learning objectives not covered by the final project.
The quiz is timed and it contributes to part of your grade as per the course description.
All aid is allowed during the quiz.

### What you must have before the quiz

* Local configuration of git/GitHub  
* Local configuration of Python with whatever IDE you're comfortable with
* Be part of the 46120-PiWE organization on GitHub (if you participated in the projects, you're fine)

### Study guide for the quiz

A student will do well on the quiz if they can:  
 * Clone a repo from a remote link
 * Make an update in a feature branch 
 * Open a well-explained pull request on GitHub
